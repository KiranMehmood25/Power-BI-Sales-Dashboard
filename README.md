# 📊 Sales Performance Dashboard (Power BI)

## 📌 Project Overview

In this project, I built a Power BI dashboard to explore and analyze sales performance using a retail dataset. The goal was to turn raw data into meaningful insights that can help understand business trends and performance.

## 🎯 Key Features

* Created KPI cards to track Total Sales, Profit, and Orders
* Built a monthly sales trend to understand performance over time
* Analyzed sales by region and product category
* Added interactive filters to explore data easily
* Highlighted key insights to support decision-making

## 🛠 Tools Used

* Power BI
* DAX (Data Analysis Expressions)
* Data Cleaning and Transformation
* Data Visualization

## 📈 Key Insight

From the analysis, I found that the West region generates the highest sales, while the Technology category performs best overall.
